# Assignment 2 — SQL Based Data Analysis

## Tools Used
MySQL Workbench, MySQL 8.0

## Database
ShopEase — an e-commerce database with 4 tables:
customers, products, orders, order_items

## Files
- setup.sql   → CREATE TABLE + INSERT statements to set up the database
- queries.sql → All 27 SQL query answers (Sections A to E)

## Topics Covered
- Section A: SELECT, Primary Keys, Constraints
- Section B: WHERE filters, Indexes, SARGable queries
- Section C: GROUP BY, SUM, COUNT, AVG, MIN, MAX, HAVING
- Section D: INNER JOIN, LEFT JOIN, 3-table JOINs, Foreign Keys
- Section E: CASE statements, ACID properties, Transactions
