# Basic Data Exploration and Cleaning using Pandas

Assignment for Celebal Excellence Internship (CEI) — Data Engineering track.

## Objective
Explore and clean a shopping dataset using Pandas: load data, inspect structure,
handle missing values, filter/select data, remove duplicates, and create a derived column.

## Files
- `Basic_Data_Exploration_and_Cleaning.ipynb` — main notebook with all steps
- `Combined_dataset.csv` — original raw dataset
- `cleaned_shopping_dataset.csv` — final cleaned output

## Steps Performed
1. Loaded CSV into a Pandas DataFrame
2. Explored data (head, tail, shape, dtypes, describe)
3. Handled missing values (filled/dropped per column)
4. Filtered rows and selected columns
5. Removed duplicate rows
6. Created a derived `total_amount` column
7. Saved cleaned dataset as a new CSV
